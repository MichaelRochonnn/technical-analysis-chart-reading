# Analysis Template

Use concise, evidence-led reports. Prefer scenario planning over one-way prediction.

## Chart Reading Report

```markdown
**Context**
- Instrument/timeframe:
- Data freshness:
- Higher-timeframe bias:

**Visible Structure**
- Trend:
- Key support:
- Key resistance:
- Channel/trendline:
- Pattern candidate:

**Confirmation**
- Volume:
- Open interest:
- Moving averages/channels:
- Oscillators/momentum:
- Divergence:

**Synthesis**
- Bullish evidence:
- Bearish evidence:
- Conflicts/missing evidence:
- Base case:

**Scenario Plan**
- Bullish trigger:
- Bearish trigger:
- Invalidation:
- Initial target/measurement:
- Risk note:
```

## Quick Signal Summary

```markdown
Bias: bullish / bearish / neutral / range-bound
Confidence: low / medium / high
Reason: 2-4 evidence bullets
Invalidation: price/structure that would disprove the view
Next confirmation: what must happen next
Risk: position sizing, stop logic, event/liquidity caveat
```

## Trade-Plan Guardrail

Before suggesting any trade setup, include:

- Entry logic, not just entry price.
- Stop/invalidation logic.
- First target and why it is plausible.
- What would make the setup invalid before entry.
- Position sizing must be risk-based, not conviction-based.
